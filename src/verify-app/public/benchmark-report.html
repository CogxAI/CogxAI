06

Key Differentiators

Capabilities that no competitor — commercial or academic — brings together today.

01

The Only Memory System That Dreams

5-phase dream cycle: consolidation, compaction, reflection, contradiction resolution, emergence. Modeled after human sleep-dependent memory consolidation. No competitor offers autonomous offline processing.

02

Active Contradiction Resolution

Proactively identifies conflicting memories via database RPC, resolves them through LLM mediation, creates resolves links, and speeds decay on the weaker memory. Reactive systems only handle conflicts at retrieval time.

03

Biologically-Inspired Decay

4 per-type exponential decay rates: episodic 0.93, procedural 0.97, semantic 0.98, self_model 0.99. Events naturally fade. Knowledge holds firm. Identity persists.

04

3 Vectors Per Memory

Per-fragment embeddings split each memory into summary, content_chunk, and tag_context vectors. Competitors generate a single embedding. CogxAI matches across three semantic layers for higher-precision recall.

05

7-Phase Recall Pipeline

Vector search, metadata filtering, candidate merging, composite scoring, entity-aware expansion, bond-typed graph traversal, type diversity injection. Most competing systems rely on 1–2 phases.

06

On-Chain Memory Commitment

Every memory is SHA-256 hashed and recorded on Base. Verifiable, immutable, tamper-evident. Proves what an agent knew and exactly when. No other memory system includes blockchain integration.
